Splitter Windows


To generate a splliter window first select "Splitter window templates" from the tools menu. The following dialog will be displayed :

The image buttons indicate the type of template that will be generated when the button is clicked. Hovering the mouse cursor over a button also pops up a tooltip with a brief description.

The default action is to test a selected template. Other options are to copy the template code to the clipboard for pasting in other source code, and to edit the code using the IDE editor. If you want to save the source code to a file you can do so from the IDE editor.
