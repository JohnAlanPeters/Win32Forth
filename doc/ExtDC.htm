ext-windc -- Extended WinDC class.

Glossary 

:class ext-windc <super windc

Extended version of the WinDC class. 

:M ClassInit:   ( -- )

Init the class 

:M Destroy:     ( -- )

Destroy the device context. 

:M SetHandle:   ( hObject -- )

Set the handle of the object.      
If the current handle of the object is valid it will be destroyed. 

:M PutHandle:   ( hObject -- )

Set the handle of the object.      
If the current handle of the object is valid it will be destroyed. 

:m ~:           ( -- )

Clean up on dispose. 

;class

End of ext-windc class
