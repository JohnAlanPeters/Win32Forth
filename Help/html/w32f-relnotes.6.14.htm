    Overview

    System Changes

    • A new console written in Forth. w32fConsole.dll is not needed anymore.
    • ForthForm has been integrated into the Win32ForthIDE.
    • Help has been totally rewritten and runs in a separate application.
    • Inter-process launching and communication via messages has been rewritten.
    • FSAVE and APPLICATION have been replaced by SAVE , and TURNKEY modified, to give easier to use saving words (see "How to convert from version 6.12.00").
    • A lot of code has been moved.
    • An improved IDE.
    • see ~\src\Compat\Evolve.f for specific to the changes between the releases 6.12.00 and 6.14.00.

    Bug fixes

    A large number of bugs have been fixed. Changes can be found at the CVS Commit List.

    New Tools

    • ShowDLLSymbols.f - Enumerates all the symbols in a DLL file.

    New classes

    • Timer-Window Class for windows that can handle timer events.
    • Ext-WinDC Extended version of the WinDC class.
    • Bitmap-DC Bitmap device context class.

    New Libraries

    • EscapedStrings.f Initially written by Stephen Pelc modified for Win32Forth by Dirk Busch.
