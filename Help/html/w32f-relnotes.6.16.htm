    Overview

    System Changes

    • Number of locals increased to 16.
    • 200X {: :} locals syntax added.
    • EXECUTE-PARSING added.
    • 200X PARSE-NAME added.
    • RECURSE now works correctly with :NONAME
    • EXIT ?EXIT and ; now work the same as EXITM ?EXITM and ;M in methods.
    • RESTORE-INPUT now repositions the file and rereads the line if input source is a file.
    • USER area increased to 8192 bytes
    • USER area moved down closer to parameter stack.
    • New user variable TASK-STATUS added to provide 32 bit flags (8 for applications), plus access words.
    • Assychronous callbacks in multi-media automatically handled.
    • EXIT, ?EXIT and and ; now work with methods as alternatives to EXITM ?EXITM and ;M.
    • Opening source files associated with an editor (WinEd, SciEdit or Win32ForthIDE) when the editor is running causes the file to display in the running instance.

    Bug fixes

    A large number of bugs have been fixed. Changes can be found at the CVS Commit List.
