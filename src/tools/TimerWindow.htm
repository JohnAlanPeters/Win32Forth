Timer-Window -- Timer-Window class.

Glossary 

:class timer-window  <super window

Timer-Window class. This class can be used for windows that should handle timer events. Only one timer per window can be used with this class. 

:m ClassInit:   ( -- )

Init the class 

:m KillTimer:   ( -- )

Destroy the timer for this window. 

:M CreateTimer: ( ms -- )

Create the timer for this window. 

:M On_Done:     ( -- )

Things to do before program termination 

:M On_Timer:    ( -- )

Thing's to do when the window recives a timer event. Default does nothing. 

;class

End of timer-window class.
