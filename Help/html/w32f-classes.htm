Classes

Windows and Dialogs

  • Generic-Window Base class for all window objects.
  • Window Class for window objects.
  • Child-Window Class for child window objects.
  • TrayWindow Class for windows that hide themself in the TrayBar when they are minimized.
  • TimerWindow Class for windows that can handle timer events.
  • DialogWindow Class for windows that contain controls.
  • MDIClientWindow MDI client control class.
  • MDIFrameWindow MDI Frame window class.
  • MDIChildWindow MDI Child window class.
  • MdiDialogWindow Class for MDI child windows that contain controls.

Controls

Control is the base class for standard window controls. All the following classes are derived from it

  • Edit controls
    • EditControl Class for text edit controls.
    • TextBox Enhanced Version of the EditControl class.
    • PassWordBox Class for text edit controls. All characters in the control are displayed as an asterisk *.
    • MultiLineTextBox Class for multi line text edit controls.
    • RichEditControl Class for rich edit controls.
  • Combo box controls
    • ComboControl Class for combo box controls.
    • ComboBox Enhanced Version of the ComboControl class.
    • ComboListBox Class for read only combo box controls.
  • List box controls
    • ListControl Class for list box controls.
    • ListBox Enhanced Version of the ListControl class.
  • Group controls
    • GroupControl Class for group controls.
  • Static controls
    • StaticControl Class for static controls.
  • Check box controls
    • CheckControl Class for check box controls.
  • Radio button controls
    • RadioControl Class for radio button controls.
  • Push button controls
    • ButtonControl Class for push button controls.
  • Other controls
    • Tab Control Class for tabulator controls.
    • Up-down Control Class for up-down controls.
    • Spinner Control Class for spinner controls.

ActiveX controls

  • AXControl Base class for ActiveX controls.
    • HTMLControl HTML control class.
    • WebServer - Simple Web Server

Dialogs defined in resource (*.res) files.

  • Dialog Class for Dialogs.
  • ModelessDialog Class for modless Dialogs.

Since Win32Forth doesn't provide any tool's to create a dialog resource you should use ForthForm to create your dialog windows instead.

GDI

  • gdiPOINT Wrapper class for a POINT struct.
  • gdiCOLORREF Wrapper class for a COLORREF struct.
  • gdiRGBQUAD Wrapper class for a RGBQUAD struct.
  • gdiSIZE Wrapper class for a SIZE struct.
  • gdiTEXTMETRIC Wrapper class for a TEXTMETRIC struct.
  • gdiBase Base class for GDI objects.
  • gdiPen Class for cosmetic pen's.
  • gdiGeometricPen Class for geometric pen's.
  • gdiBrush Base class for all brush objects.
  • gdiSolidBrush Solid brush class.
  • gdiHatchBrush Hatch brush class.
  • gdiPatternBrush Pattern brush class.
  • gdiDIBPatternBrush DIB Pattern brush class.
  • gdiFont Font class.
  • gdiBitmap Bitmap class.
  • gdiMetafile Enhanced metafile class.
  • gdiDC Device context class.
  • gdiWindowDC Window device context class.
  • gdiMetafileDC Metafile device context class.

There are some other 'old' classes in Win32Forth that are dealing with the GDI:

  • ColorObject Class for color objects.
  • ExtColorObject Class for extended color objects.
  • HatchColorObject Class for hatch color objects.
  • Font Class for fonts.
  • WinDC Device context class for windows.
  • ext-windc Extended version of the WinDC class.
  • WinPrinter Device context class for printing.
  • bitmap-dc Bitmap device context class.

All these classes are rewritten to use the GDI class library.

Other classes

  • EventClass To manage unnamed event object.
  • File Class for file I/O.
  • MultiTaskingClass For clustered tasks in objects.
  • ReadFile Class for loading/saving a complete file from/to memory.
  • SQLite Database Class for SQLite
  • WaitableTimer For high resolution waitable timers.

ADO Classes

  • ADOConnection For connecting to a database.
  • ADOCursor For manipulating recordsets.
