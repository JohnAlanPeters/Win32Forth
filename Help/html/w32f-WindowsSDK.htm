Windows API documentation

Your local link : Dowloaded win32.hlp

What is it ?

This is the Microsoft Windows documentation about all the API (Application Programming Interface) calls. It describes all the calls a program can do to the Windows System dlls in so many and different domains as Controls, Internet, Multimedia and so on. This is what we call LIBRARY and PROC in Win32Forth.

This documentation is quite necessary when creating Windows applications. Unfortunately, copyright problems (and also size problems as this is a huge documentation) don't allow us to make it available from the Win32Forth website.

How to get it ?

You have several options :

  • Buy some of the many books about API
  • Browse the MSDN website to find online documentation http://msdn2.microsoft.com
  • Find this documentation as part of some development tools (eg Borland used to include it in its C)
  • Find somewhere a downloadable version. An old version can be found at http://www.borland.com/devsupport/borlandcpp/patches/BC52HLP1.ZIP

And once downloaded ?

If you are lucky enough to download this documentation, then you can add a direct link to it in this help. You have 2 options : either you modify/activate the dummy link at the top of this page or you replace the link to this page by a local link to your download.

For the first case :

  • Open this file (help\html\w32f-WindowsSDK.htm) with a simple editor and modify/activate the link at the top of the page (after "Your local link :").

For the second case, please refer to the tools/help/help part of this help to see how to modify a link in the help summary treeview.
