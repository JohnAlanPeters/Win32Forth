The IDE Editor


The IDE Editor is a standard multiple document interface editor based on the Scintila edit control.

A lexer was written for the control which allows syntax highlighting for Forth source code.

A source file can be opened from the editor toolbar or from the File Menu. Hovering the mouse cursor over a button in the toolbar will popup a tooltip identifying the button function. Each source file will be opened in a separate window, and opened files can then be arranged or tiled to preference (from the Window menu).

The Editor supports tabbed editing i.e opened files can be displayed in tabs above the editing window. This option is selected from the preferences dialog and allows quick switching between open files. The Files tab in the TabWindow also allows switching between files.



The editor is very functional and allows easy editing of source code files as well as multi-source projects. The interface is similar to most typical Windows programs and should be mostly familiar to users who have used other editors. It is best learned by exploring and using but consider some highlights.

HTML source can quickly be previewed by pressing F10 or selecting from the "View" menu.

DexH Document Extracter - extract documentation from source code.

Search for text in files across folders. Notice that you can open a file from the resulting search listbox. This way of opening a file, as well as a couple of others (console BROWSE command, etc.) opens the file in read only mode.

You can check, in the lower left corner of the status bar, or by the color of the window, if a file is opened for editing or browsing. These colors can be set in the preferences dialog.

Optional create backup file when saving.
