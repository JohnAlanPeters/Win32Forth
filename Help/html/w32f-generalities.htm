Some generalities

Case-sensivity

Win32Forth is case insensitive.

There is one important exception however, it is for Windows procedures calls where you have to strictly follow Windows spelling : eg you have to type :

LoadLibrary GetProcAddress etc.

Forth comments

Forth traditional comments are in the form :

\ comment until end of line

( comment until ) is encountered (this can be a multiline comment)

It is sometimes useful to have other sets of comments (to transiently comment in or out some pieces of code while debugging for instance). Win32Forth offers two additionnal multiline comments :

(( comment until )) is encountered

COMMENT: comment until COMMENT; is encountered

There is also a word to comment until end of source file :

\S comment until end of file
