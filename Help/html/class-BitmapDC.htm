bitmap-dc -- Bitmap device context class.

Glossary

:class bitmap-dc <super ext-windc
Bitmap device context class. This can be used for buffered drawing.

:M ClassInit:   ( -- )
Init the class

:M Valid?:      ( -- f )
Check if it's save to use this device.

:M Fill:        { color_object \ -- }
Fill the display bitmap with a specified color.

:m FillArea:    ( color_object -- )
Fill the display bitmap with a specified color.

:m Destroy:     ( -- )
Destroys the display bitmap.

:M Init:        { width height RefDC \ -- }
Create the display bitmap and select it to our device context. Our device context will be compatible to the reference device.

:m Paint:       { ps_left ps_top ps_right ps_bottom DestDC -- }
Draw the display bitmap into the destination device condtext.

:m ~:           ( -- )
Clean up on dispose.

;class
End of bitmap-dc class
