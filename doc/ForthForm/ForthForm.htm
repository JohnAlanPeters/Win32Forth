ForthForm

The Forms Generator
For
Win32Forth




ForthForm is a form designer tool written in Win32Forth for Win32Forth. Forms designed can be saved and then compiled and floaded as a standard Forth text file or can be copied and then pasted into a program. The Forth console is also available from within the program. ForthForm has been designed to be easy to use and extensible. It is contributed to the Win32Forth community with the hope that it will be as useful to others as it has been to the author.
