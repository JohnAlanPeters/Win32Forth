Using FormPad

FormPad is a simple text editor/viewer that can be used to edit compiled forms or toolbars.


Selecting view from the toolbar dialog or for a form will display FormPad (or alternatively the Win32Forth editor if it is already opend) with the source code for the toolbar or form.

Save to Disk allows saving the displayed code to a file. This allows for a form to be saved without first having to compile it to disk, thereby eliminating the corresponding .ff file.

Exit Closes the FormPad window.
