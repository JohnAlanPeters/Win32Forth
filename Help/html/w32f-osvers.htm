Windows Versions Supported

Win32Forth supports the following operating systems;

  • Windows 95 (untested recently)
  • Windows 98
  • Windows ME (untested recently)
  • Windows NT 3.51 (untested recently)
  • Windows NT 4.0 (untested recently)
  • Windows 2000
  • Windows XP
  • Windows 2003 (not properly confirmed; only tested occasionally)
  • Windows Vista (more or less working, under development)

The preferred OS versions are 98, 2000 and XP, as a number of users regularly test with these. If you have tested under one of the untested versions, please let us know .

Win32Forth used to support OS/2 and Win32s. Neither of these is now supported. There are currently no plans to support Linux or Mac; if you're looking for Forths that support these OSes, please see the FAQ for comp.lang.forth at http://www.complang.tuwien.ac.at/forth/faq/toc.html .
