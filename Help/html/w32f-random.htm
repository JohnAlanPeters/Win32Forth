The Random Number Generator

Random numbers have many uses in programming. Win32Forth therefore provides an integer random number generator for your uses. Its use is simplicity itself : the word RANDOM ( n1 -- n2 ) returns a random number n2 between 0 and n1 - 1 for positive n1 or between n1 + 1 and 0 for negative n1.
